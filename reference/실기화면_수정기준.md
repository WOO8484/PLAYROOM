# 실기 화면 문제 요약

## 빠른 시작

- 화면 전체 배경이 푸른 회보라색으로 보여 카드와 구분이 약함
- 카드 전체에 민트·보라 그라데이션이 깔려 텍스트 영역까지 번짐
- 하단 메뉴가 반투명하여 배경이 비쳐 보임
- 페이지 이동 중 이전 화면 잔상이 남음

## 홈

- 마지막 최근 플레이 카드와 하단 메뉴가 지나치게 가까움
- 시스템 내비게이션 바 높이까지 고려한 안전 여백이 부족해 보임
- 배경과 카드 대비가 약함

## 유지할 것

- 상단 로고와 제목 구조
- 393dp 기준 화면 비율
- 카드 2개 구조와 버튼 위치
- 하단 메뉴 4개 구조
- 대표 이미지 2장 교대
- 보라색 포인트 컬러
