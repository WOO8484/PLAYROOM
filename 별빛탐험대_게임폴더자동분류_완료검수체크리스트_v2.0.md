# 별빛 탐험대 게임 라이브러리 v2.0 완료 검수 체크리스트

## 구조
- [ ] 기존 프로젝트를 직접 수정했다.
- [ ] `feature:library` 모듈이 추가됐다.
- [ ] 모든 기존 모듈이 유지됐다.
- [ ] `.github/workflows/android-build.yml`이 있다.
- [ ] `gradle-wrapper.jar`가 있다.
- [ ] ZIP 루트가 한 겹 더 감싸져 있지 않다.

## 폴더 접근
- [ ] 상단 우측 폴더 아이콘이 모든 주요 탭에서 보인다.
- [ ] `OpenDocumentTree`를 사용한다.
- [ ] persistable read URI permission을 저장한다.
- [ ] `MANAGE_EXTERNAL_STORAGE`를 요청하지 않는다.
- [ ] 권한 손실 상태를 처리한다.

## 검색
- [ ] ContentResolver/DocumentsContract로 재귀 검색한다.
- [ ] UI 스레드에서 검색하지 않는다.
- [ ] 취소가 가능하다.
- [ ] 실패 시 기존 목록을 보존한다.
- [ ] 여러 폴더를 등록할 수 있다.
- [ ] 재검색에서 중복이 생기지 않는다.

## 분류
- [ ] 확장자와 폴더 힌트를 함께 사용한다.
- [ ] 모호한 ISO/CHD/CUE/PBP를 억지로 확정하지 않는다.
- [ ] CUE/BIN이 1개 게임으로 표시된다.
- [ ] GDI 트랙이 중복 등록되지 않는다.
- [ ] M3U가 다중 디스크 대표가 된다.
- [ ] BIOS·세이브·치트·이미지 파일이 게임으로 등록되지 않는다.
- [ ] 분류 신뢰도와 이유를 저장한다.

## 데이터
- [ ] SampleGameData가 생산 Repository에서 제거됐다.
- [ ] 게임 라이브러리가 재실행 후 유지된다.
- [ ] 즐겨찾기가 유지된다.
- [ ] 빠른 시작·홈·즐겨찾기·게임리스트가 같은 데이터를 사용한다.
- [ ] 가짜 플레이 기록을 만들지 않는다.

## 커버
- [ ] 자동 타이머 커버 교대가 제거됐다.
- [ ] 선택된 게임의 실제 커버가 표시된다.
- [ ] 로컬 커버 자동 매칭이 동작한다.
- [ ] 커버가 없으면 플랫폼 대체 커버가 표시된다.
- [ ] 다른 게임의 정적 커버를 대신 사용하지 않는다.

## UI
- [ ] 기존 393dp 디자인 비율이 유지된다.
- [ ] 하단 메뉴 구조가 유지된다.
- [ ] 페이지 이동 잔상이 다시 생기지 않는다.
- [ ] 플랫폼 필터가 실제 보유 플랫폼만 표시한다.
- [ ] 게임 없는 상태가 명확하다.
