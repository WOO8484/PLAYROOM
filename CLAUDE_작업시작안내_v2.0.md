2. 게임 폴더 자동 분류·게임 라이브러리 v2.0 작업지시서

## 작업 시작

`별빛탐험대_게임폴더자동분류_대규모업데이트_실행지시서_v2.0.md`

`별빛탐험대_게임폴더자동분류_완료검수체크리스트_v2.0.md`

현재 환경에서 APK 생성이나 실제 Gradle 컴파일이 불가능하면 성공했다고 보고하지 말고,
GitHub Actions에서 빌드 가능한 전체 프로젝트 ZIP과 `BUILD_REPORT.md`를 전달하세요.

새 프로젝트 생성, WebView 전환, 샘플 데이터 유지, Gradle 버전 임의 변경,
`.github` 및 Gradle Wrapper 누락은 금지합니다.
